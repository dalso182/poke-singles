Roles de aplicación

Tico Blue (#1E3A8A)Color primario interactivo. Botones primarios, enlaces, focus rings, estados seleccionados, checkboxes activos, indicadores de tabs. Toda la capa funcional de la app.
Vault Ink (#15151A)Casi-negro con tinte púrpura sutil. Headings, body, datos. Más cálido que un negro puro, más legible.
Amber Glow (#D4941C)Indicador exclusivo de cartas raras y destacadas. Glow sutil alrededor del card, badge "DESTACADA", eyebrow accents. Nunca como botón.
Tico Red (#CE1126)Restringido a tres usos: la barra de marca superior, precios de oferta y estados de "agotado". Nunca como botón ni como acento decorativo.

★ La barra de marca

La brand bar es la firma visual de Poke-Singles: una franja de 3 píxeles con gradiente rojo → ámbar → rojo aplicada al borde superior de tarjetas hero, banners destacados, y momentos de marca. Aparece una vez por página, idealmente en la parte superior del primer bloque de contenido. Nunca como elemento decorativo aleatorio.

★ Reglas estrictas para el rojo

Tico Red (#CE1126) tiene exactamente tres usos permitidos:

① Brand bar — gradiente linear-gradient(90deg, #CE1126, #D4941C, #CE1126) aplicado en 3px al tope de tarjetas hero/featured.

② Precios de oferta — el precio rebajado se muestra en rojo, junto al precio original tachado en gris.

③ Badge AGOTADA — fondo rojo + texto blanco, esquina superior derecha del producto.

Nunca usar rojo para: botones, enlaces, focus rings, badges genéricos, fondos de sección, hover states, o cualquier acento decorativo. Esos roles son del azul.

05 — Uso del logotipo

Fondos y variantes

Fondo claro ✓ · Default
Fondo tonal ✓
Fondo oscuro ✓ · Footer/email

✓ Hacer

  • Versión clara como default — el círculo exterior negro contrasta naturalmente con superficies cream.
  • Aplicar glow ámbar sutil al hacer hover en logo de header (efecto premium).
  • Mantener proporciones exactas del círculo y los colores de la bandera CR (rojo + azul).
  • Tamaño mínimo: 40px en pantalla — los detalles de la bandera requieren claridad.
  • Espacio libre alrededor: equivalente al diámetro del círculo interior.

✕ No hacer

  • No aplicar el logo sobre las cartas — siempre sobre superficie neutra.
  • No reemplazar los colores de la bandera CR dentro del logo.
  • No usar pure white (#FFFFFF) como fondo de página — siempre Page Cream para calidez.
  • No agregar drop shadows pesados — el efecto debe ser suave y luminoso.
  • No deformar, rotar ni inclinar el logotipo.

08 — Tema Angular Material

Implementación técnica

Configuración SCSS para Angular Material 17+. Light theme con override del background a Page Cream. Primary es navy, accent es ámbar, rojo no entra en la paleta de Material — vive como CSS variable separada para los tres usos restringidos. Densidad -1 para sensación moderna y compacta.

@use '@angular/material' as mat;

// Vault Light — Modern marketplace, navy-driven
$vault-primary: mat.define-palette((
  500: #1E3A8A,  // Tico Blue — primary CTAs, links, focus
  700: #172E6E,
  contrast: (500: #FFFFFF, 700: #FFFFFF)
));

$vault-accent: mat.define-palette((
  500: #D4941C,  // Amber Glow — featured/rare indicators
  contrast: (500: #15151A)
));

$vault-warn: mat.define-palette((
  500: #B91C1C,  // Error Red — form validation only
  contrast: (500: #FFFFFF)
));

$vault-theme: mat.define-light-theme((
  color: (
    primary: $vault-primary,
    accent: $vault-accent,
    warn: $vault-warn
  ),
  typography: mat.define-typography-config(
    $font-family: 'Manrope, sans-serif',
    $headline-1: mat.define-typography-level(64px, 64px, 800, 'Manrope, sans-serif')
  ),
  density: -1
));

// Brand-level CSS variables — outside Material's color system
:root {
  --brand-red: #CE1126;       // brand bar, sale, sold-out only
  --surface-page: #FBFAF7;
  --surface-card: #FFFFFF;
}

// Override Material's default page background to warm cream
.mat-app-background {
  background-color: var(--surface-page) !important;
}

// SIGNATURE — brand bar (top of hero/featured cards)
.brand-bar {
  height: 3px;
  background: linear-gradient(
    90deg,
    var(--brand-red),
    #D4941C,
    var(--brand-red)
  );
}

// Featured card — amber glow signature
.product-card--featured {
  box-shadow: 0 0 0 1px #D4941C,
              0 12px 32px -8px rgba(212, 148, 28, 0.25);
  transition: box-shadow 200ms ease;
}

// Sale price — Tico Red, never used elsewhere for text
.price--sale {
  color: var(--brand-red);
  font-weight: 700;
}
